角标

$e^{n+2}$

如果要在左右两边都有上下标，可以用\sideset来表示：

eg: \sideset{^1_2}{^3_4}\bigotimes：$\sideset{^1_2}{^3_4}\bigotimes$。

括号

小括号与方括号

使用原始的()和[]即可。

eg: (2+3)[4+4]：$(2+3)[4+4]$。

大括号

由于大括号{}被用来分组，因此需要使用\{和\}表示大括号，也可以使用\lbrace和\rbrace来表示。

eg: \{a*b\}或者\lbrace a*b \rbrace，都会显示为${a*b}$。

尖括号

使用\langle和\rangle分别表示左尖括号和右尖括号。

eg: \langle x \rangle：$\langle x \rangle$

上取整

使用\lceil和\rceil表示。

eg: \lceil x \rceil：$\lceil x \rceil$

下取整

使用\lfloor和\rfloor表示。

eg: \lfloor x \rfloor：$\lfloor x \rfloor$

需要注意的是，原始括号并不会随着公式大小缩放。

eg: (\frac12)：$(\frac12)$。可以使用\left( …\right)来自适应的调整括号: \left( \frac12 \right)：$\left( \frac12 \right)$。

可以明显看出，后一组公式中的括号是经过缩放的。

求和与积分

求和

\sum用来表示求和符号，其下标表示求和下限，上标表示上线。

eg: \sum_1^n： $$ \sum_1^n $$

积分

\int用来表示积分符号，同样地，其上下标表示积分的上下限。

eg: \int_1^\infty： $$ \int_1^\infty $$ 与此类似的符号还有：

\prod：$\prod$

\bigcup：$\bigcup$

\bigcap：$\bigcap$

\iint：$\iint$

分式与根式

分式

分式有两种表示方法。

第一种，使用\frac，如果分子或分母不是单个字符，需要使用{}来分组。

eg:

\frac ab: $\frac ab$

\frac {a-1}{b+1}:$\frac {a-1}{b+1}$

第二种，使用\over来分隔一个组的前后两部分

eg:

{a+1\over b+1}：${a+1\over b+1}$

根式

使用\sqrt[a]b来表示。其中，方括号内的值用来表示开几次方，省略方括号则表示开方。

eg:

\sqrt[4]{\frac xy}：$\sqrt[4]{\frac xy}$

\sqrt{x^3}：$\sqrt{x^3}$

乘号

叉乘\times

eg: $\times$

Font

  • 使用\it显示意大利体（公式默认字体）：

    $\it ACDEFGHIJKLMnopqrstuvwxyz$

  • 使用\mathbb或\Bbb显示黑板粗体（黑板黑体）

    \mathbb{CHNQRZ}：$\mathbb{CHNQRZ}$

  • 使用\mathbf或\bf示黑体

    $\mathbf ABCDEFGHIJKLMnopqrstuvwxyz$

  • 使用\mathtt或\tt显示打印机字体：

    $\tt ABCDEFGHIJKLMnopqrstuvwxyz$
